# Para Executar o Projecto na sua máquina

Ter Instalado:

* Ruby 2.3.3

* Rails 5.1.6

* git clone https://github.com/UniREDENTOR/Rails.git

* cd Rails

* bundle install

* rake db:setup

* rails s

* Abrir o navegador e digitar a URL http://localhost:3000
